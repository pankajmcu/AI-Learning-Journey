    Transformers

    Transformers are a type of neural network architecture introduced in the paper
    Attention Is All You Need (Vaswani et al., 2017).
    They form the foundation of most modern large language models (LLMs), including GPT, BERT, and LLaMA.

    Why Transformers?

    Traditional models like RNNs and LSTMs read sequences step by step,
    which makes training slow and limits their ability to handle long-range context.

    Transformers use self-attention, allowing each token to consider relationships with all other tokens in the sequence, in parallel.

    Example:
    “The animal didn’t cross the street because it was too tired.”
    Self-attention links “it” → “animal”, enabling better understanding.

    Core Components

    1. Embedding Layer — converts tokens into continuous vectors.
    2. Positional Encoding — injects word order into embeddings.
    3. Self-Attention — each word looks at others to find relevant context.
    4. Multi-Head Attention — multiple attention “heads” in parallel for richer relationships.
    5. Feed-Forward Networks — add non-linear transformations.
    6. Residual Connections + Layer Norm — stabilize and speed up training.
    7. Stacked Layers — repeating blocks build depth and power.

    Encoder vs Decoder

    • Encoder stack: processes input sequences (e.g., BERT).
    • Decoder stack: generates output sequences (e.g., GPT).
    • Encoder-Decoder: full translation pipeline (e.g., original Transformer, T5).

    Visualization

    Example: Sentiment Analyzer (Python)

    from transformers import pipeline
sentiment_analyzer = pipeline("sentiment-analysis")
print(sentiment_analyzer("I love learning AI with Hugging face!"))
print(sentiment_analyzer("This is th worst movie i have ever seen."))

output:
[{'label': 'POSITIVE', 'score': 0.9997360110282898}]
[{'label': 'NEGATIVE', 'score': 0.9997711777687073}]

print(sentiment_analyzer("This is cake tastes so good."))
output:
[{'label': 'POSITIVE', 'score': 0.9998664855957031}]

    Example: Text Generator (Python)

    from transformers import pipeline
generator = pipeline("text-generation", model = "gpt2")
# Generating Text
result = generator(
    "In the future of AI,", # prompt
    max_length=50, # max tokens to generate
    num_return_sequences=1, # ho many ompletions to return
    temperature=0.7 # randomness (lower = more focussed, higher = more creative)
)
print(result[0]["generated_text"])

    Why Transformers Matter

    • Parallelizable → much faster training than RNNs/LSTMs.

    • Long-range dependencies → attention links distant words directly.

    • Scalable → billions of parameters, foundation of modern AI.
